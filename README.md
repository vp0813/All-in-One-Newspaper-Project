# All-in-One-Newspaper-Project

NewsNest is a modern **All-in-One Newspaper web application** that brings multiple news sources together on a single platform.  
It allows users to search, read, and explore news across different categories and languages in a clean and user-friendly interface.

This project is built using core web technologies such as **HTML, CSS, and JavaScript** and is designed as a **high-level academic project**.


## **There are some amazing features in this web application:**
1. **Search News** by your interest or keyword.  
2. **Language Selection** can be done (for now â€“ English and Hindi).  
3. **All news in one place**, which keeps it organised.  
4. Clean and responsive, user-friendly website.  
5. Fast and easy to access.  
6. **Saved Posts/News** section is also available for people who are preparing for competitive exams (e.g., Govt. exams) or college/school competitions.  
7. **Light or Dark Mode** is also available for reading comfort.
